 - instalações:
	cinst node-lts
	cinst yarn
	npm install -g expo-cli

 - criação de projeto:
	npm init -y
	npm install express
	npx create-react-app {nome_do_projeto}
	expo init {nome_do_projeto}

 - frontend:
	npm install react-icons
	npm install react-router-dom

- mobile:
	rodar expo no celular:
		- Baixar o expo no celular
		- no terminal (na pasta do projeto mobile): yarn start
		- Escanear o QR gerado em localhost:19002
		- Expo indica o pacote react-navigation para navegação
	- npm install @react-navigation/native
	- expo install expo-constants

 - observações:
	NPM gerencia os pacotes (instala)
	NPX executa os pacotes (só executa, exemplo, create-react-app n precisa ser instalado, só usado)
	Erros mais comuns durante a utilização do expo: https://github.com/Rocketseat/expo-common-issues
